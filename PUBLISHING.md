# Publishing AutonomyX Finder

This repo export is prepared for GitHub publishing.

## Suggested repository name

`autonomyx-finder`

## First push

Create the remote repository, then run:

```bash
git remote add origin <your-github-repo-url>
git branch -M main
git push -u origin main
```

## Optional tags

For the first release:

```bash
git tag v1.0.0
git push origin v1.0.0
```

## Configured public URLs

- homepage: `https://findr.openautonomyx.com`
- repository: `https://github.com/openautonomyx/findr`
